# Module 7 — Regulations, Records & Capstone
*Read alongside eCFR Parts 43 & 65 and AMTH General (records chapter).*

## Learning Objectives
- Apply the regulations that govern who may do maintenance and how it's approved.
- Write a **compliant maintenance record (logbook) entry**.
- Chart your path to a real **A&P** certificate.
- Complete a capstone that ties the whole course together.

## Lesson 7.1 — Who May Do What (Part 43 §43.3 / §43.7)
- **A&P mechanic** may perform & approve most maintenance, preventive maintenance, and
  minor repairs/alterations for return to service.
- **Major repairs/alterations** need approved data and are recorded on **FAA Form 337**;
  an **IA (Inspection Authorization)** approves them and performs annual inspections.
- **Certificated pilots** may do listed **preventive maintenance** (Part 43 App. A(c)) on
  their own aircraft (oil, tires, plugs, etc.).
- **Repair stations (Part 145)** and **repairmen** cover specialized/organizational work.

## Lesson 7.2 — The Documentation Chain
```
APPROVED DATA → PERFORM WORK → INSPECT → RECORD (§43.9/§43.11) → RETURN TO SERVICE (§43.7)
```
A compliant §43.9 entry contains: **(1)** description of work done (or reference to
acceptable data), **(2)** date completed, **(3)** name of the person doing the work if
other than the signer, and **(4)** signature + certificate number + kind of certificate
of the person approving return to service.

## Lesson 7.3 — Continuing Airworthiness
- **Airworthiness Directives (ADs)** — mandatory (Part 39); track compliance.
- **Service Bulletins** — manufacturer recommendations (mandatory only if AD-referenced).
- **TCDS** — the approved configuration & limits (free on FAA DRS).
- **Required inspections:** annual (all), 100-hour (for hire/instruction), progressive.

## Lesson 7.4 — Your Path to a Real A&P
1. **Route:** Part 147 AMT school **or** documented experience (30 months both ratings).
2. Pass the **written** tests (General + Airframe + Powerplant).
3. Pass the **oral + practical** with a DME.
4. Get certificated; after **3 years** you can pursue **IA**.
5. Keep learning — type courses, factory schools, avionics/NDT certifications.

## Capstone Project (do all four)
**Scenario:** A trainer aircraft came in with (a) a small skin crack near a stringer, (b) a
soft brake, and (c) a rough-running engine on one magneto.

1. **Inspect & plan:** For each squawk, name the inspection method(s) and the reference
   you'd consult (AMTH chapter, SRM, or AC 43.13-1B).
2. **Sheet-metal fix:** For the crack, specify stop-drill + a patch — give rivet size
   (dia≈3T), edge distance (2–2.5D), and pitch (4–6D) for 0.040" skin.
3. **Engine diagnosis:** The mag check shows a 250-RPM drop on the right mag (limit 175,
   max diff 50). State pass/fail and two likely causes.
4. **Write the logbook entry:** Compose a §43.9-compliant record for the crack repair,
   including all four required elements. (Use a fictitious name/cert number.)

<details><summary>Capstone answer guide</summary>
1. (a) visual + dye-penetrant, SRM/AC 43.13-1B ch.4; (b) brake system check + fluid/bleed,
AMTH Airframe landing-gear ch.; (c) magneto/ignition + spark-plug reading, AMTH Powerplant
ignition ch. 2. ~5/32" (–5) rivets (3×0.040≈0.12→next size), edge 0.31–0.40", pitch
0.63–0.94"; stop-drill crack ends first. 3. FAIL — 250 > 175 RPM limit; likely fouled/dead
plug(s) or faulty magneto/lead. 4. Entry must include description (+data ref), date,
performer name, and signature + certificate number/kind approving return to service.
</details>

## Self-Check Quiz
1. What form records a major repair/alteration?
2. What are the four required elements of a §43.9 record?
3. Are Service Bulletins mandatory? When?
4. How long as an A&P before you can get an IA?
5. Which inspection is required on ALL aircraft annually?

<details><summary>Answers</summary>
1. FAA Form 337. 2. Description/ref, date, performer name, signature+cert#+kind approving
RTS. 3. Not by themselves — only when referenced by an AD (or ops spec). 4. Three years.
5. The annual inspection.
</details>

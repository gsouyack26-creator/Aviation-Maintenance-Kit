# ✈️ How to Work on Aviation Equipment — A Free, Self-Paced Course

Built from the **aviation-maintenance skill** and the free FAA knowledge base in
`../research/`. This course takes an absolute beginner (or a cross-training industrial
maintenance tech) toward the knowledge foundation of an **FAA A&P mechanic**.

> ⚠️ **Educational only.** Completing this course does **not** certify you to maintain
> real aircraft. Legal maintenance requires FAA certification and approved data. Use this
> to *learn*, then pursue a Part 147 school or documented experience + FAA testing.

---

## Who it's for
- Newcomers curious about how aircraft are maintained.
- Industrial / MHE / controls technicians (like RME) cross-training into aviation.
- Anyone preparing to start formal A&P study.

## What you'll be able to do at the end
1. Explain how an aircraft and its engine work as systems.
2. Perform (in a training context) core tasks: inspection, safety wiring, riveting,
   fastener torque, compression testing, control rigging, corrosion treatment.
3. Read blueprints, maintenance manuals, ADs, and write a compliant logbook entry.
4. Know exactly which regulations govern who may do what, and how it's recorded.
5. Chart your path to a real **A&P** certificate.

## Prerequisites
- Basic math & mechanical aptitude. No prior aviation knowledge required.
- **Free** downloads: the three FAA AMTH handbooks + AC 43.13-1B (see Module 0).

## How to use this course
1. Do **Module 0** first (download the free books).
2. Work modules in order; General (1–2) underpins everything.
3. For each lesson, **read the linked FAA handbook chapter**, then do the lab.
4. Take the self-check quiz; if you miss items, re-read before moving on.
5. Finish with the **Module 7 capstone** (mock inspection + logbook entry).

## Assessment
- Per-module self-check quizzes (answers at the bottom of each module).
- Final capstone project in Module 7.
- (Optional) Take real FAA A&P practice written tests to benchmark.

## Grading yourself
| Score on quizzes | Meaning |
|---|---|
| 90%+ | Solid — move on |
| 70–89% | Review weak spots |
| <70% | Re-study the module |
